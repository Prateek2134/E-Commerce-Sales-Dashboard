📊 Sales Dashboard using Power BI






📌 Project Overview

This project presents an interactive Power BI dashboard created using sales and order data. The dashboard helps visualize business insights such as sales performance, orders, and customer details.

📂 Dataset Files

The dashboard is built using the following datasets:

Orders.csv – Contains order information

Details.csv – Contains additional sales and customer details

📊 Features

Interactive sales visualization dashboard

Data analysis using Power BI

Clear insights into orders, sales trends, and performance

Attractive dashboard design using custom backgrounds

🛠️ Tools Used

Microsoft Power BI

CSV Datasets

Data Visualization

🎯 Purpose

The goal of this project is to analyze sales data and present insights through interactive dashboards to support better business decisions.
